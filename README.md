## Requirements

To compile this LaTeX document, you need:
- LaTeX distribution (TeX Live, MiKTeX, or MacTeX)
- XeLaTeX engine (for font support)
- Required packages: geometry, fontspec, babel, xcolor, tcolorbox, booktabs, colortbl, hyperref, enumitem, graphicx, tikz

## Usage

1. Clone this repository
2. Compile using XeLaTeX:
   ```bash
   xelatex Documents.sty
   ```
3. The output PDF will be generated in the same directory
